## 🧠 Sobre o Projeto

O **AutoRace System** transforma a execução do sistema em uma corrida visual:

- Cada função = um carro 🏎️  
- Cada processo = uma volta 🔄  
- Cada performance = velocidade ⚡  

> 💬 "Aqui, seu código não executa… ele COMPETE."
